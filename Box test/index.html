What can you do with a box model?
display BLOCK
  • box will break onto a new line
  • width and height respected
  • padding/margin/ borders will push other elements
display INLINE
  • box will not break onto a new line
  • width and height not respected
  • horizontal padding/margin/ borders will push other elements
  • vertiacal padding/margin/ borders will not push other elements
Inner display can be:
  1. display: flex
  2. display: inline-flex
